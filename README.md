# 💎 AI-сервис предварительной оценки ювелирных изделий

Прототип интеллектуального сервиса для дистанционной оценки ювелирных изделий с использованием больших языковых моделей. Разработан в рамках хакатона "Код рождения" Школы 21, 8-10 июня 2026 года.

## 📋 О проекте

Сервис решает бизнес-проблему потери лидов: клиенты не посещают ломбард без понимания возможной суммы займа. Решение позволяет получить предварительную оценку секунд по фотографии, что увеличивает конверсию в визит и формирует квалифицированные лиды.

### Ключевые возможности MVP
- Анализ фото ювелирных изделий (375/585/750 проба) через LLM
- Расчет суммы займа и выкупа
- Оценка вероятности принятия (Высокая/Средняя/Низкая)

### Требования
- Python 3.10+
- API-ключ OpenRouter (бесплатно): [openrouter.ai](https://openrouter.ai/)

### Установка и запуск

```bash
# 1. Клонируйте репозиторий
git clone <URL_ВАШЕГО_РЕПОЗИТОРИЯ>
cd project

# 2. Установите зависимости
pip install -r requirements.txt

# 3. Создайте файл .env и добавьте ключ
echo "OPENROUTER_API_KEY=sk-or-v1-ваш_ключ" > .env

# 4. Запустите приложение (НАДЕЖНЫЙ СПОСОБ)
python -m streamlit run app.py
# или
python3 -m streamlit run app.py
```
